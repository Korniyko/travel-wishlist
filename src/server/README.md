# Travel Wishlist - Proxy Server

Proxy server for Foursquare Places API integration.

## API Endpoint

### GET /api/places
Search for places by query and location.

**Parameters:**
- `query` (required) - search term (e.g., "pizza", "museums")
- `near` (required) - location (e.g., "New York", "Paris")


## Setup

1. Install dependencies:
   ```bash
   npm install

2. Start the server::
   ```bash
   node index.js
   ```
3. Server start on  `http://localhost:5001/`.
